# Keeper proposals

Auto-generated research dossiers land here, one per run (`keeper-<date>.md`),
each opened as a pull request for review. They are **proposals, not facts** —
honestly-tiered leads from the multi-model consensus, never primary sources.

Merge a dossier to accept it into the research record; close it to reject.
See [`../KEEPER.md`](../KEEPER.md) for the full pipeline and the honesty bar.
