# Windows Font Installation Guide

Practical steps to get high-quality, professional fonts installed on Windows that work great in PowerPoint, Word, Figma, VS Code, Canva, and other tools.

## Recommended Fonts to Download First

Start with these two — they are the most versatile and already heavily used in professional environments:

| Font   | Best For                          | Recommendation                          | Direct Link |
|--------|-----------------------------------|-----------------------------------------|-------------|
| **Inter** | Body text, modern docs, slides, UI | Download this first (excellent variable font) | [Download Inter](https://fonts.google.com/specimen/Inter) |
| **Lato**  | Professional documents            | Excellent pairing with Inter            | [Download Lato](https://fonts.google.com/specimen/Lato) |

### Optional Strong Additions (once you have the first two)

- **Montserrat** — Great for headings and display type
- **Roboto** — Extremely reliable, works everywhere
- **Aleo** — Excellent for headings when you want something warmer than pure sans

## Step-by-Step: Download & Install on Windows

### 1. Download the fonts

1. Click one of the links above.
2. On the Google Fonts page, click the **Download family** button (top right).
3. The file will download as a `.zip`.

### 2. Extract the zip file

1. Right-click the downloaded `.zip` file.
2. Choose **Extract All**.
3. Pick a clean location (recommended: `Downloads\Fonts\Inter` or similar).

### 3. Install the fonts

**Easiest method (recommended):**

1. Open the extracted folder.
2. Select the `.ttf` files you want (or `Ctrl + A` to select all).
3. Right-click the selection → **Install** (current user only)  
   or **Install for all users** (best if you use PowerPoint, Word, etc. and want them available to all accounts on the machine).

**Alternative method (drag & drop):**

1. Press `Windows + R`, type `fonts`, and press Enter.
2. Drag and drop the `.ttf` / `.otf` files directly into that folder.

## After Installing

- The fonts should appear immediately in most programs (PowerPoint, Word, Canva, Figma, VS Code, etc.).
- **To verify:**
  1. Open Notepad, Word, or PowerPoint.
  2. Try typing with the new font name (e.g. “Inter” or “Lato”).

If a font doesn’t appear right away, restart the application (or your computer as a last resort).

## Pro Tips for Windows

- **Variable fonts** (Inter and Montserrat have excellent variable versions): Windows 10/11 handles these very well. You usually only need the variable file + a few key static weights.
- Install the **full family** (Regular, Medium, SemiBold, Bold, etc.) for the best experience in PowerPoint and design tools.
- For professional work, prefer WOFF2 when self-hosting on the web, but for desktop use (PowerPoint, Word, etc.), the TTF files from Google Fonts are perfect.
- Keep your downloaded font zips organized — it makes it easy to re-install on a new machine.
